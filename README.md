# Adote um pet

Projeto com front-end feito em Nextjs e API feita utilizando o framework Laravel.

API Start: {
  1- composer install
  2- npm install
  3- php artisan key:generate
  4- inicializar o xampp
}

UI STart: {
  1- npm install,
  2- npm run dev
}
